#### The App

Start with terminal logging: `rackup` and open [http://localhost:9292](http://localhost:9292)
Start with auto-restarting: `shotgun` and open [http://localhost:9393](http://localhost:9393)

#### Running specs

`rspec spec`

#### Running Rubocop

`rubocop`

#### Compass info

Compile: `compass compile`
Watch: `compass watch`
Clean: `compass clean`
